# Messaging Glossary

## Product Naming
- Product: `thesheet`
- Website/domain references: `thesheet.co`

## Preferred Language (Examples)
- specs
- selections
- project workflow
- organized
- client-ready
- decisions

## Avoid / Use Carefully
- "revolutionary"
- "game-changing"
- "seamless" (unless describing a real UX detail)
- vague "platform" language with no concrete examples

- Approved feature names
