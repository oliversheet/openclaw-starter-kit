# Voice Pack

Curated brand voice guidance for writing agents. Keep this small, opinionated, and up to date.

## Use
- Read before drafting public-facing copy
- Prefer these files over guessing tone from old drafts
- Add examples only after they are approved

## Files
- `style-guide.md` - tone, rhythm, sentence style, structure rules
- `do-dont.md` - sharp examples of preferred vs rejected writing moves
- `glossary.md` - approved terms, banned terms, and product naming conventions
